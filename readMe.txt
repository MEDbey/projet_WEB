Application emploi du temps coté serveur.

mode d'emploi :
    - télécharger l'archive sur github
    - extraire le dossier de l'archive dans le dossier www de wamp (ou htdocs pour xampp)
    - dans le dossier ressources récupérer le fichier edt.sql qui correspond a la base de donnée
    - importer ce fichier sur PhpMyAdmin
    - dans le navigateur sur l'url entrez localhost/nom_du_dossier_du_projet
    - Pour ajouter un créneau : appuyer sur le bouton correspondant et choisir dans le formulaire ce que l'on souhaite
    - Pour créer un module : appuyer sur le bouton correspondant, remplir le formulaire et envoyer
