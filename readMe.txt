

mode d'emploi :

    - télécharger l'archive sur github
    - extraire le dossier de l'archive dans le dossier www de wamp (ou htdocs pour xampp)
    - dans le dossier ressources récupérer le fichier edt.sql qui correspond a la base de donnée
    - Créer une nouvelle base de données sur PhpMyAdmin et la nommer edt puis importer le fichier edt.sql
    - dans le navigateur sur l'url entrez localhost/nom_du_dossier_du_projet

    - Pour créer un module : appuyer sur le bouton correspondant, remplir le formulaire et envoyer
    - Pour créer une matière : appuyer sur le bouton correspondant, remplir le formulaire et envoyer

    - pour modifier le nombre d'heures d'une matière : appuyer sur la case correspondante, remplir le formulaire et envoyer 
    - pour supprimer un créneau : glisser-déposer le créneau en question dans la corbeille
    - pour ajouter un créneau  à une matière et l'enlever d'une autre matière : glisser-déposer le créneau en question et le dans la case correspondante
    - pour échanger le nombre d'heures de deux matières : glisser-déposer un des deux créneaux sur l'autre
